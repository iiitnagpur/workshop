What?

A five days extensive training workshop on MATLAB, world's most widely used language for technical computing.Learn from experts in the field.

Basic Level

This workshop consists of a basic level where participants can explore essential toolboxes, functioning of preliminary math, graphics, signal processing, image processing, video processing, data import and analysis etc.

Advance Level

This workshop also consists of advance level where the participants would cover LIVE hands on training on some challenging applications in signal, image and video processing using the latest version of MATLAB.

Medical Imaging

This workshop covers the most important application which is Medical Imaging. Through this workshop you can learn to analyze ECG signals, Brain signals and many more medical applications to be explored.

ABOUT MATLAB

MATLAB is a multi-paradigm numerical computing enviroment. MATLAB combines a desktop environment tuned for iterative analysis and design processes with a programming language that expresses matrix and array mathematics directly.
MATLAB apps let you see how different algorithms work with your data. Iterate until you’ve got the results you want, then automatically generate a MATLAB program to reproduce or automate your work. MATLAB code is production ready, so you can go directly to your cloud and enterprise systems, and integrate with data sources and business systems.
MATLAB supports Model-Based Design, which is used for multidomain simulation, automatic code generation, and test and verification of embedded systems.

He has authored several papers in the refereed journals and conferences of international repute such as IEEE Transactions on Medical Imaging, Processing Letters, IET Image Processing, IET Computer Vision and Biomedical Signal Processing and Control Journal.
